## 例题8

### 题目8

使用gzip模块压缩和解压缩文件的示例(gzipfile.py)

### 答案8

```python

import sys, gzip
filename = sys.argv[0]
filenamezip = filename + '.gz'
# compress
with gzip.open(filenamezip, 'wt') as f:
    for s in open(filename, 'r'):
        f.write(s)
# uncompress
for s in gzip.open(filenamezip, 'r'):
    print(s)
```
